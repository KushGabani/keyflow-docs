title: Extract Text (OCR)
---

## Overview

The Extract Text OCR block utilizes Optical Character Recognition (OCR) technology to extract text from images or scanned documents.

## Block Inputs

| Name        | Type | Description                                               |
| ----------- | ---- | --------------------------------------------------------- |
| `file_name` | file | The file containing the image or document to perform OCR. |

## Block Outputs

| Name             | Type   | Description                                |
| ---------------- | ------ | ------------------------------------------ |
| `extracted_text` | string | The extracted text from the provided file. |

## Use Cases

1. **Digitization of Documents:** Extract text from scanned documents to make them searchable and editable.
2. **Data Extraction from Images:** Extract text from images containing textual information.
3. **Automated Data Entry:** Automate data entry processes by extracting text from documents.
4. **Text Analysis:** Analyze and process text data extracted from images for various applications.
5. **Content Management:** Manage and organize textual content extracted from images or scanned documents.
