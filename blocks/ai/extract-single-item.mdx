---
title: Extract Single Item
---

## Overview

Extract a single value for each item from a text using AI.

## Block Inputs

| Name                 | Type    | Description                                                                                                  |
| -------------------- | ------- | ------------------------------------------------------------------------------------------------------------ |
| `items`              | list    | A list of items that accepts `name` and `description` fields for each item.                                  |
| `additional_context` | string  | Additional context or information to enhance the extraction process.                                         |
| `cache_response`     | boolean | When enabled, it saves you AI calls by returning the same response if the inputs to the LLM has not changed. |

## Block Outputs

<Note>
  The output handles of type `string` are dynamically added as the items
  provided in the list.
</Note>

## Use Cases

1. **Product Information Extraction:** Extracting specific details such as price or availability from a list of products.
2. **Event Details Extraction:** Retrieving essential information like date, time, and location from a list of events.
3. **Contact Information Extraction:** Extracting contact details such as name, email, and phone number from a list of contacts.
4. **Inventory Management:** Extracting key attributes like quantity and location from a list of inventory items.
5. **Document Parsing:** Extracting metadata or key points from a list of documents.
