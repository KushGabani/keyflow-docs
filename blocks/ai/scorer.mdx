## Overview

Assigns a numerical score to an input based on specific criteria using the power of LLM.

## Block Inputs

| Name                    | Type    | Description                                                                             |
| ----------------------- | ------- | --------------------------------------------------------------------------------------- |
| `item`                  | string  | The subject, entity or text that needs to be scored.                                    |
| `criteria`              | string  | The description of rules or scoring criteria to use for evaluation.                     |
| `include_justification` | boolean | When enabled, the model also returns reasoning or explaination for the evaluated score. |

## Block Outputs

| Name            | Type   | Description                                             |
| --------------- | ------ | ------------------------------------------------------- |
| `score`         | number | Reflects the assessment based on the provided criteria. |
| `justification` | string | Detailed explanation for the score (if any).            |

## Use Cases

1. **Comparing Products:** Evaluate and compare different products based on defined criteria.
2. **Content Quality Review:** Quantitatively assess the quality of content such as articles, videos, or images.
3. **Decision-Making Based on Standards:** Use clear, numerical feedback to guide decisions based on defined standards.
4. **Record-Keeping Purposes:** Generate justifications for assessments for understanding and record-keeping.
