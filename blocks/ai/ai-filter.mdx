Processes data by applying a natural language condition to each input.

**Functionality:** Uses powerful language-model algorithms to interpret the condition and input, deciding whether the input meets the specified condition.

**When To Use:** Can be used for sorting or categorizing textual data according to specific criteria expressed in natural language.

### Inputs & Outputs

| I/O    | Name            | Type   | Description                                                                                                                                                                                  |
| ------ | --------------- | ------ | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| input  | condition       | String | The natural language condition you want to be applied to the inputs.                                                                                                                         |
| input  | input           | String | The content you want the filter to be applied to.                                                                                                                                            |
| output | filtered_output | String | The result of the AI Filter's evaluation. If the input data meets the specified condition, you will receive the same input data here. If it does not meet the condition, this will be empty. |

### Use Cases

- **Filtering Customer Inquiries:** Identifying and separating inquiries based on specific criteria expressed in natural language.
- **Data Categorization:** Sorting textual data according to specified conditions.
- **Automated Decision Making:** Making dynamic decisions based on textual content in complex automation processes.
