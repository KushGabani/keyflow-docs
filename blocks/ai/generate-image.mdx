title: Generate Image
---

## Overview

Generate images from prompt texts or image description using SOTA AI Models.

## Block Inputs

| Name         | Type     | Default   | Description                                                                                               |
| ------------ | -------- | --------- | --------------------------------------------------------------------------------------------------------- |
| `prompt`     | string   |           | Input description or prompt text for the AI model.                                                        |
| `num_images` | number   | 1         | Determines the quantity of images to generate Maximum is 10.                                              |
| `image_size` | selector | 512 x 512 | Resolution of the generated images. Can be one of the following: `256 x 256`, `512 x 512`, `1024 x 1024`. |

## Block Outputs

| Name         | Type      | Description                                                                                  |
| ------------ | --------- | -------------------------------------------------------------------------------------------- |
| `image_urls` | string[ ] | File paths for all the generate images which can be used as an input for other image blocks. |

## Use Cases

1. **Content Creation:** Provides visually captivating images for social media, marketing, or educational purposes.
2. **Conceptual Visualization:** Useful for creating visuals to convey a specific concept or idea.
3. **Exploring AI Creativity:** Offers a convenient and innovative solution for exploring the intersection of AI and creativity.
4. **Quick Visual Representation:** Ideal for scenarios requiring quick and creative visual representation without the need for human artists.
