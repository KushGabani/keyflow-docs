Condenses extensive text into concise, informative summaries using GPT, which processes text in manageable sections.

**Useful for:** Use this block when you need to distill lengthy articles, reports, or documents into a shortened form to grasp the main ideas quickly without going through the entire text. It is particularly useful for getting a quick understanding of large volumes of information.

### Inputs and Outputs

| I/O    | Feature | Type   | Description                                                                                  |
| ------ | ------- | ------ | -------------------------------------------------------------------------------------------- |
| input  | text    | string | The extensive piece of text you wish to summarize.                                           |
| output | summary | string | The condensed version of the input text, which retains the essential points and information. |

### Use Cases

- **Summarizing Articles and Reports:** Quickly grasp the main ideas of lengthy articles and reports.
- **Distilling Large Documents:** Get a quick understanding of large volumes of information.
- **Grasping Essential Points:** Condense extensive text to retain essential points and information.
- **Understanding Lengthy Text:** Get a quick understanding of complex and lengthy text.
