## Overview

Condenses extensive text into concise, informative summaries using GPT, which processes text in manageable sections.

## Block Inputs

| Name             | Type     | Description                                                                                                  |
| ---------------- | -------- | ------------------------------------------------------------------------------------------------------------ |
| `text`           | string   | The extensive piece of text you wish to summarize.                                                           |
| `model`          | selector | LLM Model to be used for inference. Can be one of the following supported models.                            |
| `cache_response` | boolean  | When enabled, it saves you AI calls by returning the same response if the inputs to the LLM has not changed. |

## Block Outputs

| Name      | Type   | Description                                                                                  |
| --------- | ------ | -------------------------------------------------------------------------------------------- |
| `summary` | string | The condensed version of the input text, which retains the essential points and information. |

## Use Cases

1. **Summarizing Articles and Reports:** Quickly grasp the main ideas of lengthy articles and reports.
2. **Distilling Large Documents:** Get a quick understanding of large volumes of information.
3. **Grasping Essential Points:** Condense extensive text to retain essential points and information.
4. **Understanding Lengthy Text:** Get a quick understanding of complex and lengthy text.
