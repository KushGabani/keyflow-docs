## Overview

Convert text input into spoken audio using AI.

## Block Inputs

| Name           | Type   | Description                                              |
| -------------- | ------ | -------------------------------------------------------- |
| `text`         | string | The text content that needs to be converted into speech. |
| `new_filename` | string | The desired name for the resulting MP3 file.             |

## Block Outputs

| Name            | Type | Description                                                                           |
| --------------- | ---- | ------------------------------------------------------------------------------------- |
| `mp3_file_name` | file | The name of the created MP3 file, which contains the audio version of the input text. |

## Use Cases

1. **Educational Purposes**: For creating audio lessons or lectures from written materials.
2. **Accessibility**: Assisting visually impaired users by providing audio versions of text.
3. **Content Creation**: Generating voiceovers for videos, presentations, or podcasts.
4. **Language Learning**: Assisting in language learning by providing audible examples of text.
5. **Personalized Alerts and Notifications**: Creating custom audio messages for use in apps or websites.
