## Overview

Convert spoken audio into text using AI.

## Block Inputs

| Name         | Type | Description                                                     |
| ------------ | ---- | --------------------------------------------------------------- |
| `audio_file` | file | The audio file containing the spoken content to be transcribed. |

## Block Outputs

| Name            | Type   | Description                                     |
| --------------- | ------ | ----------------------------------------------- |
| `transcription` | string | The text transcription of the input audio file. |

## Use Cases

1. **Transcription Services**: Convert recorded interviews, meetings, or lectures into text for documentation purposes.
2. **Accessibility**: Provide text versions of spoken content for individuals with hearing impairments.
3. **Language Learning**: Generate transcripts of spoken language for language learners to study and practice.
4. **Automatic Captioning**: Automatically generate captions for videos and other multimedia content.
5. **Searchable Archives**: Create searchable text archives of spoken content for easy reference and retrieval.
