## Overview

Classifies text data into predefined categories using AI.

## Block Inputs

| Name                    | Type     | Description                                                                                                  |
| ----------------------- | -------- | ------------------------------------------------------------------------------------------------------------ |
| `categories`            | list     | Information about all the categories: A list of `category_name` and `category_description` for the AI model. |
| `input`                 | string   | The input string to be categorized.                                                                          |
| `include_justification` | boolean  | When enabled, the model also returns reasoning or explaination for the AI's response.                        |
| `model`                 | selector | LLM Model to be used for inference. Can be one of the following supported models.                            |
| `cache_response`        | boolean  | When enabled, it saves you AI calls by returning the same response if the inputs to the LLM has not changed. |

## Block Outputs

| Name                | Type   | Description                                                       |
| ------------------- | ------ | ----------------------------------------------------------------- |
| `selected_category` | string | The chosen category name after the categorization process.        |
| `justification`     | string | Explanation provided by the AI for category selection. (optional) |

## Use Cases

1. **Customer Feedback Analysis:** Automatically sort feedback into different themes for analysis.
2. **Product Categorization:** Organize products based on descriptions.
3. **Document Organization:** Categorize and label documents by topic.
4. **Interpretability:** Understand the reasoning behind AI's category selection.
