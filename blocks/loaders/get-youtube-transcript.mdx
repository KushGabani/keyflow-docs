Retrieves transcripts from YouTube videos.

- **Content Extraction**: Focuses on spoken content of videos.
- **Accessibility Aid**: Useful for creating subtitles or supporting hearing-impaired individuals.
- **Content Analysis**: Assists in analyzing spoken material in educational or informative videos.

### Inputs & Outputs

| I/O    | Name       | Type   | Description                                                                                 |
| ------ | ---------- | ------ | ------------------------------------------------------------------------------------------- |
| input  | video_link | String | The complete URL of the YouTube video, e.g., "https://www.youtube.com/watch?v=jNQXAC9IVRw". |
| output | transcript | String | The text of the YouTube video's transcript, dialogue only, without timestamps.              |

### Use Cases

- **Text Quoting**: Extracts spoken content for quoting in text form.
- **Reading vs. Watching**: Enables reading the video's content instead of watching it, offering a faster, more accessible alternative.
- **Subtitle Creation**: Facilitates the creation of subtitles for accessibility.
- **Video Content Analysis**: Suitable for analyzing spoken content in lectures, interviews, and presentations.
- **Translation Preparation**: Provides a written basis for translating video content into other languages.
