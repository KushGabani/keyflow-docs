## Overview

Retrieves transcripts of a YouTube video.

## Block Inputs

| Name        | Type   | Explanation                                                                                 |
| ----------- | ------ | ------------------------------------------------------------------------------------------- |
| `video_url` | string | The complete URL of the YouTube video, e.g., "https://www.youtube.com/watch?v=jNQXAC9IVRw". |

## Block Outputs

| Name         | Type   | Explanation                                     |
| ------------ | ------ | ----------------------------------------------- |
| `transcript` | string | The retrieved transcript of the youtube video". |

### Use Cases

1. **Text Quoting**: Extracts spoken content for quoting in text form.
2. **Reading vs. Watching**: Enables reading the video's content instead of watching it, offering a faster, more accessible alternative.
3. **Subtitle Creation**: Facilitates the creation of subtitles for accessibility.
4. **Video Content Analysis**: Suitable for analyzing spoken content in lectures, interviews, and presentations.
5. **Translation Preparation**: Provides a written basis for translating video content into other languages.
