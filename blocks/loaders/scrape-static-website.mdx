Automates information collection from web pages.

- **Content Extraction**: Focuses on retrieving readable text from web pages, excluding code or scripts.
- **PDF Compatibility**: Skillfully handles and extracts text from PDF documents linked in web URLs.

### Inputs & Outputs

| I/O    | Name            | Type   | Explanation                                                                                                                  |
| ------ | --------------- | ------ | ---------------------------------------------------------------------------------------------------------------------------- |
| input  | url             | String | The web address of the page for information collection, e.g., "https://keyflow.space/". Must start with http:// or https://. |
| output | website_content | String | The main readable text gathered from the visited website, devoid of any coding or scripting elements.                        |

### Use Cases

- **Content Monitoring**: Ideal for tracking changes in web content over time.
- **Research Material Collection**: Efficiently gathers research data from various websites.
- **Archiving Web Content**: Useful for archiving readable content from web pages.
- **Digital Project Integration**: A valuable tool for incorporating web content into digital projects.
- **Time-Saving in Web Browsing**: Automates the process of visiting and reading web pages, saving significant time and effort.
