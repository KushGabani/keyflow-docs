Read contents of a file as utf-8 text that can be further processed in your flows. Think of this block as your go-to tool for any task where you need to pull text out of files. You can also extract texts from file that are generated by the flow at runtime.

<Check>
  **Supported file types include** .pdf, .json, .csv, .txt, .html, .json
  and.docx
</Check>

## Block Inputs

| Name        | Type | Description               |
| ----------- | ---- | ------------------------- |
| `file_name` | file | The file to read as text. |

## Block Outputs

| Name            | Type   | Description                             |
| --------------- | ------ | --------------------------------------- |
| `file_contents` | string | Contents of the file as a utf-8 string. |

## Use Cases

1. **Analyzing Reports:** Need numbers or details from a report? It's got you covered.
2. **Database Updates:** Moving information from a CSV file to a database.
3. **Form Processing:** Read form responses from various formats.
4. **Archiving Documents:** Save important information from contracts or invoices.
