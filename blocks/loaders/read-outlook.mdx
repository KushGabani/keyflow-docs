## Overview

Read any email from your Microsoft Outlook account.

Itâ€™s particularly beneficial for managing email-based workflows efficiently, ensuring no important communication is missed.
This block also allows reading only the mails with specific folders of your outlook account.

<Note>
  To run this block, you will first need to authenticate your Microsoft Outlook
  account with Keyflow. Navigate to your
  [credentials](https://keyflow.space/settings/credentials) page to connect.
</Note>

## Block Inputs

| Name     | Type   | Default | Explanation                                                             |
| -------- | ------ | ------- | ----------------------------------------------------------------------- |
| `folder` | string | INBOX   | Identifies a specific folder or sub-folder within your outlook account. |

## Block Outputs

| Name            | Type      | Explanation                                   |
| --------------- | --------- | --------------------------------------------- |
| `sender`        | string[ ] | Email ID of the sender.                       |
| `receiver`      | string[ ] | Email ID of the receiver                      |
| `email_subject` | string[ ] | Subject of the email                          |
| `email_body`    | string[ ] | Body of the email (can be HTML or plain text) |

## Use Cases

1. **Automated Customer Support:** Responding automatically to customer queries via email.
2. **Order and Invoice Processing:** Handling order or invoice emails for business operations.
3. **Email Monitoring:** Keeping track of specific email labels for urgent updates or notifications.
4. **Email Archiving:** Regularly backing up emails from designated labels.
5. **Data Extraction:** Pulling information from emails for analysis or reporting.
