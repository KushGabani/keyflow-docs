title: Current DateTime
---

## Overview

Get a formatted string of the current DateTime based on the timzeone.

## Block Inputs

| Name            | Type     | Explanation                                                                                    |
| --------------- | -------- | ---------------------------------------------------------------------------------------------- |
| `format`        | selector | Format of the current date.                                                                    |
| `custom_format` | string   | A custom format string using `yyyy`, `mm`, and `dd` to create a personalized formatted string. |
| `timezone`      | selector | Sets the current DateTime to fetch in a specific timezone                                      |

## Block Outputs

| Name        | Type   | Explanation                               |
| ----------- | ------ | ----------------------------------------- |
| `date_time` | string | Formatted string of the current DateTime. |
