## Overview

Extract all database properties and it's type of a notion database.

Itâ€™s particularly beneficial for generating content for your notion database by LLMs. When database properties are provided within the context, LLMs generate accurate shape of the content which can then instantly be used to add to your notion database.

<Note>
  To run this block, you will first need to authenticate your Notion account
  with Keyflow. Navigate to your
  [credentials](https://keyflow.space/settings/credentials) page and connect
  your Notion.
</Note>

## Block Inputs

| Name          | Type   | Explanation                                |
| ------------- | ------ | ------------------------------------------ |
| `database_id` | string | Link or unique ID of your notion database. |

## Block Outputs

| Name                | Type   | Explanation                                                                          |
| ------------------- | ------ | ------------------------------------------------------------------------------------ |
| `properties_json`   | object | JSON containing the property name and it's corresponding field type in the database. |
| `raw_json_response` | object | Raw JSON response from the Notion API.                                               |

<Note>
  The block outputs are variable for Read Notion Page because as it depends on
  the notion database and the properties it has. When you set the `database_id`
  input, All your notion database properties will automatically be added as
  individual handles which you can then connect to other blocks directly to use
  that data.
</Note>
