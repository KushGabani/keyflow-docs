## Overview

Read data from a specific sheet of your Google Docs.

It streamlines the process of extracting data from Google Docs, making it instrumental for any requirement that involves the automated retrieval textual data.

<Note>
  To run this block, you will first need to authenticate your Google account
  with Keyflow. Navigate to your
  [credentials](https://keyflow.space/settings/credentials) page and connect
  your Google Docs using "Read & Write Google Docs"
</Note>

## Block Inputs

| Name              | Type   | Explanation                           |
| ----------------- | ------ | ------------------------------------- |
| `file_link_or_id` | string | Link or unique ID of your Google Doc. |

## Block Outputs

| Name          | Type         | Explanation                              |
| ------------- | ------------ | ---------------------------------------- |
| `doc_content` | string[ ][ ] | The data retrieved from your Google Doc. |

## Use Cases

1. **Data Migration**: Move data from Google Docs to another target service of your choice.
2. **Text Analysis:** Evaluate or processing Google Docs content automatically using natural language.
3. **Change Monitoring:** Keep an eye on document modifications by regularly reading the content.
