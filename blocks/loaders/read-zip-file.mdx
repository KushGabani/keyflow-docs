## Overview

Read contents of a zip file as utf-8 text that can be further processed in your flows. You can also extract texts from zip file that are generated by the Zip File Writer block at runtime. This operator is perfect for situations where you need to dive into a zip file and work with the files inside.

<Check>
  **Supported file types include** .pdf, .json, .csv, .txt, .html, .json and
  .docx inside the zip files.
</Check>

## Block Inputs

| Name            | Type | Description              |
| --------------- | ---- | ------------------------ |
| `zip_file_name` | file | The zip file to extract. |

## Block Outputs

| Name            | Type      | Description                                                    |
| --------------- | --------- | -------------------------------------------------------------- |
| `file_names`    | string[ ] | A list of file names found inside the zip file.                |
| `file_contents` | string[ ] | A list of actual file contents of each file as a utf-8 string. |

## Use Cases

1. **Document Processing:** Read large amounts of file and process it inside a flow.
2. **File Organization:** Helps in sorting and understanding what's inside a big zip file without manually opening it.
