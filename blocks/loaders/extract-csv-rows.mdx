Reads and converts CSV file contents into a string array.

### Inputs & Outputs

| I/O    | Feature       | Type               | Explanation                                                                  |
| ------ | ------------- | ------------------ | ---------------------------------------------------------------------------- |
| input  | csv_file_name | File               | Specifies the CSV file to be processed.                                      |
| input  | has_headers   | Boolean (Optional) | Indicates if the CSV file includes column headers at the top of each column. |
| output | csv_rows      | String Array       | An array where each element represents a row from the CSV file as a string.  |

### Use Cases

- **Data Analysis Automation**: For scenarios needing automated extraction and analysis of CSV data.
- **Workflow Efficiency**: Integrating CSV row extraction into broader automation workflows, especially where data is transferred to other processes or systems.
- **Time-Saving**: Beneficial in handling large or numerous CSV files, reducing manual data processing.
- **Error Reduction**: Eliminates the risk of manual errors in data transcription or extraction processes.
