## Overview

The Flatten List block flattens a nested list structure into a single-level list.

## Block Inputs

| Name         | Type   | Description                      |
| ------------ | ------ | -------------------------------- |
| `input_list` | any[ ] | The nested list to be flattened. |

## Block Outputs

| Name   | Type   | Description                                 |
| ------ | ------ | ------------------------------------------- |
| `list` | any[ ] | The flattened list containing all elements. |

## Use Cases

1. **Nested List Flattening:** Flatten a list of lists to create a single-level list, combining all elements.
2. **Data Transformation:** Convert hierarchical data structures into a flat format for easier processing.
3. **Data Aggregation:** Combine multiple levels of nested lists into a single list for aggregation or analysis.
4. **List Simplification:** Streamline list structures by removing nesting, making them easier to work with and understand.
