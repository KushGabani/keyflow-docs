## Overview

Processes input text line-by-line and identify the lines most closely related to a specified search query.
This is a valuable tool for quickly retrieving the most relevant information from within a large text without manual reading.

## Block Inputs

| Name          | Type   | Description                                                                                           |
| ------------- | ------ | ----------------------------------------------------------------------------------------------------- |
| `num_results` | number | The maximum number of search results to return. If not specified, default is set to return 5 results. |
| `query`       | string | The text phrase you want to look for within the provided text input.                                  |
| `text`        | string | The larger body of text where the node will look for the query.                                       |

## Block Outputs

| Name            | Type   | Description                                                      |
| --------------- | ------ | ---------------------------------------------------------------- |
| `search_result` | string | The lines of the input text that are most relevant to the query. |
