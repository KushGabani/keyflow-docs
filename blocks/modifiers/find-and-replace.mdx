## Overview

Search through a provided piece of text and replace specific words or phrases with user-provided replacements.

## Block Inputs

| Name           | Type   | Description                                                              |
| -------------- | ------ | ------------------------------------------------------------------------ |
| `replacements` | list   | Specifies the words to be found and their respective replacement words.  |
| `input`        | string | The string on which the find and replace operations will be carried out. |

## Block Outputs

| Name            | Type   | Description                                                                   |
| --------------- | ------ | ----------------------------------------------------------------------------- |
| `output_string` | string | The resulting string after all instances of the specified words are replaced. |

## Use Cases

1. **Editing Documents:** Replace outdated terms with updated terminology.
2. **Data Cleanup:** Replace unwanted characters or correct spelling errors.
3. **Customized Messages:** Replace placeholder tags with actual content, such as names or dates.
