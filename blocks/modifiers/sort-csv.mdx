## Overview

Efficiently organizes rows within a CSV file by sorting based on specified column values. It simplifies data restructuring tasks without manual sorting, enhancing data analysis, visualization, and report creation.

## Block Inputs

| Name            | Type    | Description                                 |
| --------------- | ------- | ------------------------------------------- |
| `csv_file_name` | file    | The name of the CSV file to be sorted.      |
| `column_index`  | number  | The index of the column to be sorted on.    |
| `has_headers`   | boolean | Indicates whether the CSV file has headers. |
| `reverse_sort`  | boolean | Defines the order of the sort.              |

## Block Outputs

| Name                   | Type   | Description                                                          |
| ---------------------- | ------ | -------------------------------------------------------------------- |
| `sorted_csv_file_name` | string | The name of the sorted CSV file that has been generated by the node. |

## Use Cases

1. **Prioritizing or Analyzing Data:** Organize data according to certain numeric values or text.
2. **Data Preparation for Reports:** Prepare data before creating reports, where sorted information is easier to interpret.
3. **Data Cleaning and Categorization:** Tasks that require the data to be in a specific order.
