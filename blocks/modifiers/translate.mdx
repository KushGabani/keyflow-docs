title: Translate
---

## Overview

Translate text from one language to another target language.

## Block Inputs

| Name              | Type     | Description                                                                                                                                                                               |
| ----------------- | -------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `text`            | string   | The text to be translated.                                                                                                                                                                |
| `target_language` | selector | The target language for translation. For a list of supported languages, refer to the [Google Cloud Translate Languages Documentation](https://cloud.google.com/translate/docs/languages). |

## Block Outputs

| Name              | Type   | Description                                           |
| ----------------- | ------ | ----------------------------------------------------- |
| `translated_text` | string | The translated text in the specified target language. |

## Use Cases

1. **Multilingual Communication:** Translate text to communicate effectively with speakers of different languages.
2. **Global Content Localization:** Adapt content for international audiences by translating it into various languages.
3. **Cross-Cultural Collaboration:** Facilitate collaboration across language barriers by providing translated text.
4. **Language Learning:** Assist in language learning by providing translations for better comprehension.
