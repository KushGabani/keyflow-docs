## Overview

Access only a single element from a list.

## Block Inputs

| Name    | Type   | Description                                                    |
| ------- | ------ | -------------------------------------------------------------- |
| `list`  | any[ ] | List of elements                                               |
| `index` | number | Index number of the element to access. Indexing starts with 0. |

## Block Outputs

| Name      | Type | Description                                    |
| --------- | ---- | ---------------------------------------------- |
| `element` | any  | The value available at that index of the list. |

## Use Cases

1. **Nitpick values:** Sometimes you need to access just the filename from the entire file path. Read Element List can help you access only the element needed from the entire list.
