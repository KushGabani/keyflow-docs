### Quick Overview

- **What It Does:** Merges two separate pieces of text using a user-defined pattern.
- **Input Types:** String, Template
- **Useful For:** Standardizing text combinations for various automation and text processing tasks.

### Inputs and Outputs

| Feature         | Type     | Description                                                                                     |
| --------------- | -------- | ----------------------------------------------------------------------------------------------- |
| format          | Template | Indicates how to combine the two input strings. Uses `{input1}` and `{input2}` as placeholders. |
| input1          | String   | Represents the first string to be combined. (Optional)                                          |
| input2          | String   | Represents the second string to be combined. (Optional)                                         |
| combined_string | String   | The final text resulting from merging input1 and input2 according to the format.                |

### Use Cases

- **Personalized Messages:** Create custom messages by combining different text elements.
- **Report Formatting:** Standardize the combination of data for reports.
- **Automated Workflows:** Construct structured text in automation processes.
- **Textual Information Integration:** Bring together textual information from separate sources in a consistent manner.
