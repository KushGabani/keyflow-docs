# Chunk Text

### Quick Overview

- **What It Does:** Splits a long string of text into smaller, more manageable pieces based on the specified number of characters.

### **Inputs and Outputs**

| Feature     | Type    | Description                                                      |
| ----------- | ------- | ---------------------------------------------------------------- |
| chunk_size  | Integer | Specifies the number of characters for each text piece.          |
| text        | String  | The body of text to be split into smaller pieces.                |
| text_chunks | List    | List of text pieces, each with a specified number of characters. |

- **Use Cases:**
  - When working with limitations on text size, such as API requests that only accept a certain number of characters at a time.
  - Displaying text to users in a limited space and ensuring it fits properly.
  - Text analysis, where analyzing smaller sections can be more manageable or comparing chunks of text against each other.
  - Providing a summary or preview of a text where only a specific character count is allowed.
