## Overview

Format a given string based on the specified formatting condition. This block is your go-to tool for standardizing textual data by converting text to lowercase or uppercase, capitalizing the first letter of each word, removing extra spaces, or shortening the string to a defined number of tokens.

## Block Inputs

| Name             | Type   | Description                                                                                                                              |
| ---------------- | ------ | ---------------------------------------------------------------------------------------------------------------------------------------- |
| `value`          | string | The string that you want to format.                                                                                                      |
| `formatter`      | string | The formatting you want to apply to the given string. Options include: To Lowercase, To Uppercase, To Propercase, Trim Spaces, Truncate. |
| `max_num_tokens` | number | The number of tokens (words or characters) to which you want to limit your string. Required if using the "Truncate" formatter.           |

## Block Outputs

| Name              | Type   | Description                                |
| ----------------- | ------ | ------------------------------------------ |
| `formatted_value` | string | The string after the specified formatting. |

## Use Cases

1. **To Lowercase:** For uniformity in text, like converting email addresses or usernames.
2. **To Uppercase:** For making titles or acronyms stand out.
3. **To Propercase:** For ensuring each word starts with a capital letter, often used for names or titles.
4. **Trim Spaces:** For cleaning up the text by removing leading and trailing spaces.
5. **Truncate:** For shortening textual data to a specific length, such as summarizing content or adhering to character limits in databases or other systems.
