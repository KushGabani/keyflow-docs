title: Send Whatsapp
---

## Overview

Send a message to a specified recipient via WhatsApp using the provided sender number. This block facilitates direct communication via WhatsApp, allowing you to send messages programmatically.

## Block Inputs

| Name        | Type   | Description                                               |
| ----------- | ------ | --------------------------------------------------------- |
| `send_from` | string | The phone number of the sender.                           |
| `send_to`   | string | The phone number of the recipient to receive the message. |
| `message`   | string | The content of the message to be sent.                    |

## Block Outputs

| Name             | Type   | Description                                                                                                 |
| ---------------- | ------ | ----------------------------------------------------------------------------------------------------------- |
| `message_status` | string | Indicates the status of the message sending operation, whether it was successful or encountered any errors. |

## Use Cases

1. **Personal Messaging:** Send personal messages or updates to friends or family members via WhatsApp.
2. **Business Communication:** Facilitate direct communication with customers or clients for customer support, inquiries, or updates.
3. **Notification Alerts:** Use WhatsApp messages for sending notification alerts or reminders to users about events, appointments, or deadlines.
4. **Marketing Campaigns:** Incorporate WhatsApp messaging into marketing campaigns for reaching out to potential customers or promoting products/services.
5. **Automated Workflows:** Integrate WhatsApp messaging into automated workflows or systems for sending alerts, notifications, or reports based on specific triggers or events.
