title: Read JSON
---

## Overview

Extract specific pieces of information from a JSON object. Simplify extracting specific values from a JSON object for data analysis, reporting, or data pipeline preprocessing.

## Block Inputs

| Name   | Type   | Description                                                            |
| ------ | ------ | ---------------------------------------------------------------------- |
| `keys` | string | List of comma-separated keys for extracting values from the JSON data. |
| `json` | object | The JSON from which values will be extracted.                          |

## Block Outputs

| Name          | Type   | Description                                               |
| ------------- | ------ | --------------------------------------------------------- |
| `json_values` | string | Consolidates all the extracted values from the JSON data. |

## Use Cases

1. **Data Extraction:** Useful for retrieving specific pieces of data from a JSON structure.
2. **Data Analysis:** Simplifies the process of extracting and analyzing data from JSON sources.
3. **Reporting:** Can be used to extract relevant information for reporting purposes.
4. **Data Pipeline Preprocessing:** Helps in the preprocessing stage of a data pipeline by extracting required data from JSON.
