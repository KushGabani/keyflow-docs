---
title: Write JSON
---

## Overview

Modify JSON data by adding or updating key-value pairs. This block allows modifying a JSON object without the need for manual editing or custom code.

## Block Inputs

| Name         | Type   | Description                                                                                                                       |
| ------------ | ------ | --------------------------------------------------------------------------------------------------------------------------------- |
| `key`        | string | List of comma-separated keys representing the keys in the JSON object where the value will be added or updated.                   |
| `json`       | object | The original JSON data to update. If not provided, creates a new JSON object.                                                     |
| `json_value` | string | List of comma-separated values that will be added or updated in the JSON data, can be a new or updated value for an existing key. |

## Block Outputs

| Name           | Type   | Description                                                                                                                  |
| -------------- | ------ | ---------------------------------------------------------------------------------------------------------------------------- |
| `updated_json` | object | The new version of the original JSON, now containing the changes made by adding or updating the value for the specified key. |

## Use Cases

1. **Updating Configuration Settings:** Easily insert new values into a JSON object or update existing values for configuration purposes.
2. **Modifying API Response Data:** Ensure that response data from an API contains up-to-date information by using this block.
3. **Dynamic Updating of JSON Content:** Simplify tasks involving the manipulation of JSON content within automated processes.
