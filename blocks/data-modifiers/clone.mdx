## Overview

Create an array with repetitive elements based on a specified input element and array size.

## Block Inputs

| Name                  | Type   | Description                                                                                                    |
| --------------------- | ------ | -------------------------------------------------------------------------------------------------------------- |
| `element`             | any    | The element to be cloned throughout the resulting array.                                                       |
| `array_size_to_match` | any[ ] | Optional array input that determines the size of the new array. The size is based on the length of this array. |
| `number_of_copies`    | number | Specifies the number of times the input element will be repeated in the array.                                 |

## Block Outputs

| Name                | Type   | Description                               |
| ------------------- | ------ | ----------------------------------------- |
| `array_of_elements` | any[ ] | The resulting array with cloned elements. |

### Use Cases

1. **Array Initialization:** Efficiently initialize arrays with specific sizes and predefined elements.
2. **Programming Tasks:** Useful for programming tasks requiring arrays with repetitive elements.
3. **Testing Purposes:** Facilitates testing code that necessitates arrays with specific lengths and identical elements.
