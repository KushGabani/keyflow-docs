---
title: Read From Google Sheets
---

## Overview

Read data from a specific sheet of your Google Sheets.

It streamlines the process of extracting data from Google Sheets, making it a valuable tool for tasks that regularly retrieve data from spreadsheets and saves time by automating data retrieval.

<Note>
  To run this block, you will first need to authenticate your Google account
  with Keyflow. Navigate to your
  [credentials](https://keyflow.space/settings/credentials) page and connect
  your Google Sheets using "Read & Write Google Sheets"
</Note>

## Block Inputs

| Name              | Type     | Explanation                                                                                         |
| ----------------- | -------- | --------------------------------------------------------------------------------------------------- |
| `file_link_or_id` | string   | Link or unique ID of your Google Sheet.                                                             |
| `sheet_name`      | string   | Specific sheet name within the file; defaults to the first sheet.                                   |
| `reader_mode`     | selector | Mode of data retrieval: `Row-by-Row`, `Column-by-Column`, `Specified Row`, or `Specified Column`.   |
| `row_index`       | number   | Index of the row to read., starting at 0; Required when using "Specified Row" as reader_mode.       |
| `column_name`     | string   | Name of the column to read. Starts with 'A'. Required when using "Specified Column" as reader_mode. |

## Block Outputs

| Name         | Type         | Explanation                            |
| ------------ | ------------ | -------------------------------------- |
| `sheet_data` | string[ ][ ] | The data retrieved from Google Sheets. |

## Use Cases

1. **Automated Report Generation:** Extracting data for financial reports, surveys, etc.
2. **Scheduled Record Updates:** Updating records based on sheet data.
3. **Data Migration**: Move data from Google Sheets to another target service of your choice.
