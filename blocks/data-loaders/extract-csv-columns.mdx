---
title: Extract CSV Column
---

## Overview

Extract a specific column from your CSV file.

## Block Inputs

| Name            | Type     | Explanation                                                                                                  |
| --------------- | -------- | ------------------------------------------------------------------------------------------------------------ |
| `csv_file_name` | file     | The file to read as text.                                                                                    |
| `column_index`  | number   | Index of the column to be read. Indexing starts at 0.                                                        |
| `has_headers`   | checkbox | If true, it skips the first line of the csv file considering them as headers and reads from the second line. |

## Block Outputs

| Name         | Type      | Explanation                                           |
| ------------ | --------- | ----------------------------------------------------- |
| `csv_column` | string[ ] | The specified column data extracted from the CSV file |

## Use Cases

1. **Data Analysis**: Isolating a column for detailed examination or statistical analysis.
2. **Workflow Integration**: Utilizing column data in subsequent stages of an automation process.
3. **Report Generation**: Extracting specific data for reporting purposes or list creation.
4. **Simplifying Data Access**: Facilitating data extraction for users without extensive data handling expertise, focusing on a single column rather than the full file.
