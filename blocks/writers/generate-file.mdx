## Overview

The Generate File block converts provided text into a digital file in one of several common format. It offers versatility and convenience for various document creation needs.

<Check>**Supported file types include** .pdf, .csv, .txt, .md, and .docx</Check>

## Block Inputs

| Name            | Type     | Description                                                                                             |
| --------------- | -------- | ------------------------------------------------------------------------------------------------------- |
| `file_name`     | string   | The desired name of the file to be created (without the file extension).                                |
| `file_type`     | selector | The format for the generated file. Can be one of the following: `.pdf`, `.docx`, `.txt`, `.csv`, `.md`. |
| `file_contents` | string   | The text content to be included in the file.                                                            |

## Block Outputs

| Name                  | Type   | Description                                                          |
| --------------------- | ------ | -------------------------------------------------------------------- |
| `generated_file_name` | string | The full name of the generated file, including its format extension. |

## Use Cases

1. **Drafting Formal Documents:** Create formal documents in various formats to meet specific requirements.
2. **Creating Reports:** Generate professional reports in PDF or DOCX formats for presentations or sharing.
3. **Saving Notes:** Quickly save and organize important notes as TXT files for easy reference.
4. **Organizing Data Tables:** Convert tabular data into CSV format for further analysis or integration with other tools.
