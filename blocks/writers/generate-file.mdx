## Overview

- **What It Does:** Takes the text you provide and turns it into a digital file in one of several common formats (.pdf, .docx, .txt, .csv).
- **File Types Supported:** PDF, DOCX, TXT, CSV.
- **Useful For:** Creating digital documents for various purposes, such as drafting formal documents, creating reports, saving notes, or organizing data tables.

### Inputs and Outputs

| Name          | Type   | Description                                                                                |
| ------------- | ------ | ------------------------------------------------------------------------------------------ |
| file_name     | string | The name you want to give to the file you are creating. Do not include the file extension. |
| file_type     | enum   | The file format you want to create, such as .pdf, .docx, .txt, .csv.                       |
| file_contents | string | The actual content or text that you want to put inside your file.                          |

| Name                | Type   | Description                                                                 |
| ------------------- | ------ | --------------------------------------------------------------------------- |
| generated_file_name | string | The full name of the file that was created, including its format extension. |

### Use Cases

- **Drafting Formal Documents:** Creating formal documents in various formats.
- **Creating Reports:** Generating reports in PDF or DOCX format.
- **Saving Notes:** Quickly saving important notes as a TXT file.
- **Organizing Data Tables:** Converting data into a CSV format for analysis.

Essentially, the Generate File node is your go-to tool whenever you need to create digital documents for your computer projects.
