## Overview

- **What It Does:** Automates the process of adding data to a Google Sheets file.
- **Inputs:**
  | Name | Type | Description |
  | ------------------------------------ | --------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------- |
  | #data | List of strings | Data to be written to the Google Sheet. |
  | use_existing_google_sheet (Optional) | Boolean | Indicates whether to write to an existing Google Sheet (true) or to create a new one (false). |
  | sheet_link_or_id (Optional) | Link or ID | Link to the existing Google Sheet in Google Drive or the sheet's ID. |
  | sheet_name (Optional) | String | The name of the specific sheet within the Google Sheet file where data will be written. |
  | writer_mode | String | Specifies how the data will be written to the sheet. Options include "Add New Row", "Add New Column", "Add to Specified Row", or "Add to Specified Column". |
  | row_index (Optional) | Number | Index of the row when adding data to a specified row, starting from 0. |
  | column_index (Optional) | Number | Index of the column when adding data to a specified column, starting from 0. |
- **Output:**
  | Name | Type | Description |
  | ---------- | ---- | -------------------------------------------------------- |
  | sheet_link | Link | The link to the Google Sheet where the data was written. |
- **Use Cases:**
  - Automate data entry into a Google Sheet.
  - Create new sheets or update existing ones with new data programmatically.
  - Organize collected data into structured tables within Google Sheets.
  - Share real-time data outputs by writing to a centrally accessible Google Sheet.
