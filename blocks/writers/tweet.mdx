## Overview

Send tweets directly from your Twitter account through an automated process using the Tweet block.

<Note>
  To run this block, you will first need to authenticate your Twitter (Now X)
  account with Keyflow. Navigate to your
  [credentials](https://keyflow.space/settings/credentials) page and connect
  your account.
</Note>

## Block Inputs

| Name              | Type    | Description                                                                                                     |
| ----------------- | ------- | --------------------------------------------------------------------------------------------------------------- |
| `tweet_text`      | string  | The text content of the tweet you want to send.                                                                 |
| `remove_hashtags` | boolean | Determines whether to remove hashtags from the tweet text before posting. (Optional, default is set to `false`) |

## Block Outputs

| Name        | Type   | Description                             |
| ----------- | ------ | --------------------------------------- |
| `tweet_url` | string | The URL of the tweet posted on Twitter. |

## Use Cases

1. **Real-time Updates:** Share real-time updates, announcements, or news on Twitter.
2. **Event Promotion:** Promote upcoming events or webinars to your Twitter followers.
3. **Content Sharing:** Share blog posts, articles, or multimedia content with your Twitter audience.
4. **Engagement:** Interact with your followers by initiating discussions or asking questions through tweets.
5. **Marketing Campaigns:** Launch marketing campaigns or product promotions on Twitter to reach a wider audience.
